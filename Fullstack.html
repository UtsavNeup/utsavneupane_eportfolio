What is Full Stack?

Full stack means building all parts of a website:

  1. The front (what users see)
  2. The back (how it works)
  3. The database (where data is stored)

Frontend - The Face

This is what you see in your browser. It includes:

  • Design and layout
  • Buttons and forms
  • Everything you click on

Tools used:

HTML CSS JavaScript

Backend - The Brain

This runs on the server and does the hard work:

  • Logs users in
  • Saves your data
  • Does calculations

Tools used:

Node.js Python PHP

Database - The Memory

This stores all the information:

  • User accounts
  • Website content
  • Settings

Tools used:

MySQL MongoDB PostgreSQL
