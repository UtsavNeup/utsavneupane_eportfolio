What is HTML?

HTML is the building blocks of every website. It creates the structure of web pages.

Simple HTML Example:

<p>This is a paragraph</p>

<a href="page.html">Click me</a>

Basic HTML Facts:

  • HTML stands for HyperText Markup Language
  • Uses tags like <h1>, <p>, <img>
  • Tags come in pairs - opening and closing
  • Works with CSS (for style) and JavaScript (for action)

Most Common Tags:

<h1>Big Heading</h1>

<p>Paragraph text</p>

<a href="#">Link</a>

<img src="photo.jpg">

<div>Box container</div>
